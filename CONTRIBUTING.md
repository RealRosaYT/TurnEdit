# Contribute TurnEdit
TurnEdit is an text editor.  
Written in C#.  
If know .NET C# and Windows Forms, you can contribute.
## Required for contribute
1. .NET SDK 9.0
## How to Contribute
1. [Download Source](https://github.com/suzuki3932/TurnEdit/releases/).
2. Edit code.
3. Build the project.
4. Send pull request.
